## データセット一覧

このリポジトリでは、以下のデータを公開しています。元データを加工し、機械可読性と再利用性を高めた形式で再配布します。

| ファイル名 | 内容 | レコード数 |
|---|---|---:|
| `shika_corecurriculum_r4_goals.csv` | 歯学教育モデル・コア・カリキュラム（令和4年度改訂版）の学修目標 | 651 |
| `shika_corecurriculum_h28_goals.csv` | 歯学教育モデル・コア・カリキュラム（平成28年度改訂版）の学修目標 | 542 |
| `shika_kokushi_r5_kijun.csv` | 歯科医師国家試験出題基準（令和5年版）の出題項目 | 1,378 |

### コード体系について

各データセットの階層コードは正規化されています。各レベルの `code` は自身のローカル識別子のみを保持し、親のコードは含みません。フルパスが必要な場合は `id` フィールドを参照してください。

```
例（R4版）:
  id = "A-1-1-1"
  dai_code = "A"    ← 大項目のコード
  chu_code = "1"    ← 中項目のコード
  sho_code = "1"    ← 小項目のコード
  goal_code = "1"   ← 学修目標のコード
```

## json形式について

ビューワーアプリを用意しています（下記参照）。ビューワーアプリではcsvを直接パースして内部データとしてもよかったのですが、敢えて、一旦jsonに変換してそれを用いています。その副産物としてのjson形式もviewerディレクトリ内に配置していますので、それをご利用いただくことも可能です。

## 1. shika_corecurriculum_r4_goals.csv

### 概要

令和4年度改訂版の歯学教育モデル・コア・カリキュラムに記載された学修目標を構造化したデータ

### 元データ

- **出典**: 文部科学省「歯学教育モデル・コア・カリキュラム（令和4年度改訂版）」
- **URL**: <https://www.mext.go.jp/a_menu/koutou/iryou/mext_00009.html>

### 列定義

| 列名 | 説明 |
|---|---|
| `id` | 一意識別子。階層の有無により長さが変わる: `大項目-中項目-目標番号`（小項目なし）、`大項目-中項目-小項目-目標番号`、`大項目-中項目-小項目-サブ小項目-目標番号`、`大項目-中項目-小項目-サブ小項目-サブサブ小項目-目標番号`（例: `B-1-1`, `A-1-1-1`, `A-3-1-1-1`, `E-5-3-4-1-1`） |
| `dai_code` | 大項目コード（例: `A`） |
| `dai_title` | 大項目名（例: `生命科学`） |
| `chu_code` | 中項目コード（例: `1`） |
| `chu_title` | 中項目名（例: `生命の分子基盤`） |
| `sho_code` | 小項目コード（例: `1`）。小項目なしの場合は空文字 |
| `sho_title` | 小項目名（例: `身体を構成する組織と器官`）。空文字の場合あり |
| `sub_sho_code` | サブ小項目コード（例: `1`）。サブ小項目なしの場合は空文字 |
| `sub_sho_title` | サブ小項目名（例: `上皮組織と皮膚・粘膜系`）。サブ小項目なしの場合は空文字 |
| `sub_sub_sho_code` | サブサブ小項目コード（例: `1`）。サブサブ小項目なしの場合は空文字 |
| `sub_sub_sho_title` | サブサブ小項目名（例: `固定性補綴装置(クラウンブリッジ)による治療`）。サブサブ小項目なしの場合は空文字 |
| `goal_code` | 目標番号（例: `1`） |
| `goal_text` | 学修目標の本文 |
| `source_pdf` | 参照PDFファイル名 |
| `source_page` | PDFページ番号 |

### 階層構造

```
大項目 (dai)  例: E 診察・診断と治療技能
 └─ 中項目 (chu)  例: 5 基本的治療手技
     └─ 小項目 (sho)  例: 3 高頻度治療
         └─ サブ小項目 (sub_sho)  例: 4 歯質と歯の欠損の治療  ※ない場合あり
             └─ サブサブ小項目 (sub_sub_sho)  例: 1 固定性補綴装置(クラウンブリッジ)による治療  ※ない場合あり
                 └─ 学修目標 (goal)  例: 1 固定性補綴治療に必要な情報を収集し、…
```

フルパスは `id` で表現: `E-5-3-4-1-1`（小項目なしの場合: `B-1-1`）

## 2. shika_corecurriculum_h28_goals.csv

### 概要

平成28年度改訂版の歯学教育モデル・コア・カリキュラムに記載された学修目標を構造化したデータ

### 元データ

- **出典**: 文部科学省「歯学教育モデル・コア・カリキュラム（平成28年度改訂版）」
- **URL**: <https://www.mext.go.jp/a_menu/koutou/iryou/mext_00009.html>

### 列定義

| 列名 | 説明 |
|---|---|
| `id` | 一意識別子。階層の有無により長さが変わる: `大項目-中項目-丸数字`（小項目なし）、`大項目-中項目-小項目)-丸数字`（学習目標項目なし）、`大項目-中項目-小項目)-(学習目標項目)-丸数字`（例: `A-3-①`, `A-1-1)-①`, `C-3-4)-(1)-①`）。小項目番号には `)` を付す |
| `dai_code` | 大項目コード（例: `A`） |
| `dai_title` | 大項目名 |
| `chu_code` | 中項目コード（例: `1`） |
| `chu_title` | 中項目名 |
| `sho_code` | 小項目コード（例: `1`）。小項目なしの場合は空文字 |
| `sho_display_code` | 小項目の表示用コード（例: `1)`）。空文字の場合あり |
| `sho_title` | 小項目名。空文字の場合あり |
| `goal_section_code` | 学習目標項目コード（例: `1`）。学習目標項目なしの場合は空文字 |
| `goal_section_display_code` | 学習目標項目の表示用コード（例: `(1)`）。空文字の場合あり |
| `goal_section` | 学習目標項目名（例: `上皮組織と皮膚・粘膜系`）。空文字の場合あり |
| `goal_code` | 目標コード（数字: `1`, `2`, ...） |
| `goal_display_code` | 目標の表示用コード（丸数字: `①`, `②`, ...） |
| `goal_text` | 学修目標の本文 |
| `source_pdf` | 参照PDFファイル名 |
| `source_page` | PDFページ番号 |

### 階層構造

```
大項目 (dai)  例: C 生命科学
 └─ 中項目 (chu)  例: 3 人体の構造と機能
     └─ 小項目 (sho)  例: 4 身体を構成する組織と器官
         └─ 学習目標項目 (goal_section)  例: (1) 上皮組織と皮膚・粘膜系  ※ない場合あり
             └─ 学修目標 (goal)  例: ① 上皮組織の形態、機能及び分布を説明できる。
```

フルパスは `id` で表現: `C-3-4)-(1)-①`（学習目標項目がない場合: `A-1-1)-①`、小項目もない場合: `A-3-①`）

## 3. shika_kokushi_r5_kijun.csv

### 概要

令和5年版の歯科医師国家試験出題基準に記載された出題項目を構造化したデータ

### 元データ

- **出典**: 厚生労働省「歯科医師国家試験出題基準（令和5年版）」
- **URL**: <https://www.mhlw.go.jp/stf/shingi2/0000163627_00002.html>

### 列定義

| 列名 | 説明 |
|---|---|
| `id` | 一意識別子（`領域プレフィックス-大項目-中項目-小項目`） |
| `domain_code` | 領域コード（例: `hisshu`） |
| `domain_title` | 領域表示名（例: `必修`） |
| `dai_code` | 大項目コード（例: `1`） |
| `dai_title` | 大項目名 |
| `chu_code` | 中項目コード（例: `ア`） |
| `chu_title` | 中項目名 |
| `sho_code` | 小項目コード（例: `a`） |
| `sho_title` | 小項目名 |
| `remarks` | 備考（空文字の場合あり） |
| `source_pdf` | 参照PDFファイル名 |
| `source_page` | PDFページ番号 |

### domain 値一覧

| domain_code | domain_title |
|---|---|
| `hisshu` | 必修 |
| `souron1` | 総論Ⅰ 保健・医療と健康増進 |
| `souron2` | 総論Ⅱ 正常構造と機能、発生、成長、発達、加齢変化 |
| `souron3` | 総論Ⅲ 病因、病態 |
| `souron4` | 総論Ⅳ 主要症候 |
| `souron5` | 総論Ⅴ 診察 |
| `souron6` | 総論Ⅵ 検査 |
| `souron7` | 総論Ⅶ 治療 |
| `souron8` | 総論Ⅷ 歯科材料と歯科医療機器 |
| `kakuron1` | 各論Ⅰ 成長・発育に関連した疾患・病態 |
| `kakuron2` | 各論Ⅱ 歯・歯髄・歯周組織の疾患 |
| `kakuron3` | 各論Ⅲ 顎・口腔領域の疾患 |
| `kakuron4` | 各論Ⅳ 歯質・歯・顎顔面欠損と機能障害 |
| `kakuron5` | 各論Ⅴ 配慮が必要な高齢者・有病者・障害者等に関連した疾患・病態・予防ならびに歯科診療 |

### IDプレフィックス

| プレフィックス | 領域 |
|---|---|
| `H` | 必修 (hisshu) |
| `S1`〜`S8` | 総論Ⅰ〜Ⅷ (souron1〜souron8) |
| `K1`〜`K5` | 各論Ⅰ〜Ⅴ (kakuron1〜kakuron5) |

### 階層構造

```
領域 (domain)  例: 必修
 └─ 大項目 (dai)  例: 1 医の倫理と歯科医師のプロフェッショナリズム
     └─ 中項目 (chu)  例: ア 医の倫理
         └─ 小項目 (sho)  例: a 患者の人権と医療
```

フルパスは `id` で表現: `H-1-ア-a`

### 原本の誤植と修正

| id | 小項目名 | 原本の誤り | 修正内容 |
|---|---|---|---|
| `S1-2-エ-d` | 成人歯科保健活動 | 小項目コードが `c` と誤記 | `d` に修正 |

原本PDF（000920686.pdf p.3）では `S1-2-エ-c` として `成人歯科保健活動` が記載されていますが、直前の `健康増進法に基づく保健事業` が `c` であるため、本データセットでは `d` に修正しています。

本アプリはコード署名されていないため、初回起動時にOSのセキュリティ機能によりブロックされることがあります。
